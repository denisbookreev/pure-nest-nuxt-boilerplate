## Installation

```bash
$ yarn install
```

## Running the server app

```bash
# development
$ yarn start:server

# preview production
$ yarn start:client:prod
```

## Running the client app

```bash
# development
$ yarn start:client

# watch mode
$ yarn start:server:dev

# production mode
$ yarn start:server:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
